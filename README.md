# 📘 CPP_Practice

A collection of C++ programs written as part of a structured learning journey. This repository covers fundamental to intermediate C++ concepts through hands-on practice.

---

## 📁 Files

| File | Description |
|------|-------------|
| `hello.cpp` | Hello World — the classic first program |
| `character.cpp` | Character handling and operations |
| `conversion.cpp` | Data type conversion techniques |
| `conversion1.cpp` | Extended conversion examples |
| `bubblesort.cpp` | Bubble Sort algorithm implementation |
| `linearsearch.cpp` | Linear Search algorithm implementation |

---

## 🚀 How to Run

```bash
g++ filename.cpp -o output
./output
```
